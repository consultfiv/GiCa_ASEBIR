Guía de instalación: RStudio y Python (Anaconda)
🧪 RStudio
1. Descargar e instalar R
Enlace de descarga: https://cran.r-project.org/

Pasos:

Accede al enlace.

Descarga la versión de R correspondiente a tu sistema operativo (Windows, macOS o Linux).

Ejecuta el archivo descargado e instala R siguiendo las instrucciones del instalador.

2. Descargar e instalar RStudio
Enlace de descarga: https://posit.co/download/rstudio-desktop/

Pasos:

Descarga la versión gratuita de RStudio Desktop.

Instálala siguiendo los pasos del instalador.

3. Comprobación
Abre RStudio.

En la consola escribe:

r
Copiar
Editar
print("¡R está funcionando correctamente!")
Si ves el mensaje ¡R está funcionando correctamente!, la instalación fue exitosa.

🐍 Python (Anaconda)
1. Descargar e instalar Anaconda
Enlace de descarga: https://www.anaconda.com/download

Pasos:

Accede al enlace.

Descarga la versión de Anaconda correspondiente a tu sistema operativo (Windows, macOS o Linux).

Ejecuta el instalador y sigue los pasos recomendados.

2. Abrir el entorno de trabajo
Abre Anaconda Navigator o Jupyter Notebook.

Acepta cualquier mensaje inicial o configuración predeterminada.

3. Comprobación
Abre un nuevo cuaderno de Jupyter Notebook o una consola.

Escribe:

python
Copiar
Editar
print("¡Python está funcionando correctamente!")
Si ves el mensaje ¡Python está funcionando correctamente!, la instalación fue exitosa.
